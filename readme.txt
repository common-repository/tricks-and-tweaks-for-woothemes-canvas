Tricks and Tweaks for WooThemes Canvas is an uber-flexible toolkit that allows you to modify Canvas even FURTHER. All from the touch of a button.

== Description ==

Tricks and Tweaks for WooThemes Canvas enables you to modify the Header, Footer, Navigation, Posts & Pages, Sliders, WooFramework and Miscellaneous features through an options panel. If you want to have access to some of the most commonly requested Canvas mods without getting your hands dirty with code, then this is the plugin you need!

== Installation ==

Minimum Requirements
WordPress 3.8 or greater
PHP version 5.2.4 or greater
MySQL version 5.0 or greater
Automatic installation
Automatic installation is the simplest way to install this plugin as WordPress.org transfers all of the files it needs within your web browser.
To do this, log in to your WordPress dashboard, go to the Plugins menu and click “Add New”.
In the search field type “tricks tweaks canvas” and click “Search Plugins”. When you find this plugin, you can install it by clicking “Install Now”.
Manual installation
Manual installation involves downloading this plugin and uploading it to your webserver with an  FTP application. The WordPress codex contains instructions on how to do this here.

Updating
Automatic updates can sort this out for you. Please make sure that you keep regular back-ups of your site.
To update the plugin manually, go to the Plugins menu, find this plugin and click “Update”. (Option only shows if there is an update available)
